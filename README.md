# Primeira-Igreja-Reformada-do-Capao-Redondo
⛪ Site da Primeira Igreja Reformada do Capão Redondo Este é o repositório do site institucional da Primeira Igreja Reformada do Capão Redondo (PIRCR). O objetivo deste projeto é fornecer informações claras sobre a liturgia, horários de cultos, avisos da comunidade e canais de contato para os membros e visitantes.
